# Yova Auto — Local API

Node.js + Express + SQLite backend for the garage management app.

## Setup

```bash
cd backend
npm install
```

## Run

```bash
npm run dev
```

API: **http://localhost:3001**

Default admin login (seeded on first run):

- Email: `admin@yovaauto.co.uk`
- Password: `admin`

Database file: `backend/data/yova-auto.db`
